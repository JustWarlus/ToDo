## Installation
First clone the repository:  
`  
git clone https://github.com/JustWarlus/ToDo.git  
`  
Second install npm and parcel: 
`  
cd ToDo  
`  
`  
npm install  
`  
`  
npm install parcel  
`  
Next we need to run project:  
`  
npm run dev  
`
